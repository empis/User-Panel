User-Panel
==========
<b>Features</b>

<b>Requirements</b>
-Mysql database
-Bukkit plugin Authme-optional
-Bukkit plugin Refsys-optional
<b>Mysql database</b>
1. For login, credits, referals
Table name: authme
-username-text
-password-text
-referal-text
-kredity-int
-nalakany-int
-admin-int
2. For codes
Table name: code
-code-text
-usage-text
-player-text
-time-int
<b>Installation</b>
1. Copy to webserver
2. Create mysql databases
3. Change mysql.php and mysqlcode.php
4. Enjoy
